【SC Calendar】
スケジュールカレンダーの無料配布版です。
使い方の詳細はこちらに記述しています。
https://wm-web-se-pg.com/publish-program/sc-calendar/

ホームページ上で営業日などを表示したいと思うことはありませんか？
そういったときに手軽に使えるカレンダーです。

【主要機能】
・js呼び出しでカレンダーを表示可能、HTMLに簡単に設置できます。
・カレンダー曜日表記に日本語、英語を選択可能
・PC版表示の場合、月曜始まりと日曜始まりの選択可能
・カレンダーの表示範囲を前後24カ月まで設定可能
・カレンダーのタイトルをつけることができます。
・色を変えたい日付はワンクリックでOK！

【その他機能】
・任意の日付の色を変えることができます。※デフォルトは休日の色をイメージしています。
・レスポンシブ対応 PCはテーブル表示、スマートフォンはリスト表示に自動で変換します。
　※レスポンシブ対応はCSSのため、スマートフォンでリスト表示にしたくない場合は該当箇所を削除してください。
・CSSで大きさや色などを自由に調整できます。
・データ保持にCSVファイルを使用しています。データベースは不要です。
・管理画面は簡易ログイン機能を提供しています。


【登録方法】
管理画面でカレンダー日付をクリックすると色が変わります。

【稼働環境】
PHP5.5以上（5.5、5.6、7.0、7.1、7.2で確認済み）

Microsoft Edge / IE11 / FireFox / Chrome / Opera / Android8.1.0で動作確認済み
